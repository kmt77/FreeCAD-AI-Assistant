# FreeCAD AI Assistant

An **AI-powered FreeCAD plugin** that generates and executes Python code from natural language instructions.

## ✨ Features
- 📝 Prompt box: describe what you want FreeCAD to do.
- 🔑 Enter and save your OpenAI API key directly inside FreeCAD.
- 🤖 AI-powered code generation (via OpenAI API).
- 🛠 Editable code editor.
- ▶️ Run button to execute generated Python code directly in FreeCAD.
- 📜 Output console with error reporting.

## 📦 Installation
1. Download the ZIP and extract it into your FreeCAD `Mod` directory:
   - Linux: `~/.FreeCAD/Mod/`
   - Windows: `C:/Users/<username>/AppData/Roaming/FreeCAD/Mod/`
   - Mac: `~/Library/Preferences/FreeCAD/Mod/`
2. Restart FreeCAD.
3. Select **AI Code Assistant** from the Workbench selector.

## 🔑 API Key Setup
1. Open the AI Code Assistant panel in FreeCAD.
2. Enter your **OpenAI API key** in the text box.
3. Click **Save API Key** (the key is stored in FreeCAD’s preferences).
4. Now you can generate AI-powered FreeCAD Python code!

## 🧪 Without AI
If no API key is set, the plugin will generate example demo code instead.
