
# FreeCAD AI & Static Shape Assistant

**Author:** Kmtec Ltd  
**Contact:** contact@kmtec.co.uk

This FreeCAD workbench allows beginners to create **basic shapes** and **AI-generated FreeCAD Python code** using OpenAI GPT models.

## Features
- Static shapes: Cube, Sphere, Cylinder, Cone, Torus, Wedge, Box, Circle, Rectangle, Polygon
- AI-generated FreeCAD Python code based on text prompts
- Runnable Python code inside FreeCAD
- Beginner-friendly and educational

## Requirements
- FreeCAD 0.20+
- Python 3.12+
- `openai` Python library if using AI features (`pip install openai`)

## Installation
1. Copy the folder `FreeCAD-AI-Shape-Assistant` to your FreeCAD `Mod` directory.
2. Rename the main macro file to `FreeCAD_AI_Shape_Assistant.FCMacro`.
3. Restart FreeCAD.
4. Run the macro from the Macro menu.
